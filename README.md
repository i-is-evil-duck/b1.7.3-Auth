### Minecraft Beta 1.7.3 Authenticate Plugin

BetaAuth is a Minecraft Beta 1.7.3 plugin for craftbukkit tested on Bukkit-0.0.0-980-g4ed23b1-b1060jnks

When a new player joins they will be prompted to run `/register <password>` or `/login <password>`. Players are teleported to a holding position at Y=100 until authenticated. The player can rejoin without logging in as long as they are on the same IP address (IP cache time can be configured, default 12hrs).

Unregistered attempts to move are blocked with a reminder message.
